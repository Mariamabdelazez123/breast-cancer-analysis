# Breast Cancer Diagnosis ML Project 

Machine Learning system for breast cancer diagnosis with **97.37% accuracy**

##  Project Overview
A comprehensive data analysis and machine learning project for breast cancer detection and classification, achieving outstanding performance in tumor classification.

##  Models Performance
- **Random Forest**: 97.37% 
- **SVM**: 97.37%   
- **Logistic Regression**: 96.49% 

##  Project Structure
- `breast_cancer_analysis.ipynb` - Main analysis notebook with data exploration and ML models
- `requirements.txt` - Python dependencies and libraries
- `README.md` - Project documentation

##  Technologies Used
- Python • Jupyter Notebook
- Pandas • NumPy (data manipulation)
- Scikit-learn (machine learning)
- Matplotlib • Seaborn (visualization)
